# personalfinance-prime-directive
React implementation of [/r/personalfinance Prime Directive](https://www.reddit.com/r/personalfinance/wiki/commontopics) with the help of [react-flow](https://reactflow.dev/) library. [Live Demo](https://andynguyen.dev/personalfinance-prime-directive/)

Based of the [flowchart](https://i.imgur.com/lSoUQr2.png) created by [/u/atlasvoid](https://www.reddit.com/user/atlasvoid) for the [/r/personalfinance](https://www.reddit.com/r/personalfinance/) subreddit.

## Run locally
```
npm install
npm run build
```
